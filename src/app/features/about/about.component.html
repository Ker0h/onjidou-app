Welcome to Onjidou!

Created by: Yannick Willems, 2128086

Onjidou will be an app where users can create and manage translations to the lyrics of songs, to help them study various languages. Users will be able to add songs and their lyrics, and be able to add multiple translations or transcriptions so they can be read in tandem with the original text.
For example:

Mandarin Chinese with Pinyin:
yīnyuè
音乐
Mandarin Chinese with Zhuyin/Bopomofo:
ㄧㄣㄩㄝˋ
音乐
Mandarin Chinese with Pinyin and Zhuyin:
yīnyuè
ㄧㄣㄩㄝˋ
音乐
Japanese with Romaji:
ongaku
音楽
Japanese with Hiragana:
おんがく
音楽
Japanese with both Hiragana and Romaji:
ongaku
おんがく
音楽
